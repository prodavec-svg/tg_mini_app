## Мини-апп для тестирования биржевых гипотез

Есть десять активов:
* 5 акций
* 5 облигаций

Пользователь выбирает сколько каждого актива купить (не больше 1000 каждого) и на какой срок выставить гипотезу.  
Пример:
Пользователь выбирает 3 акции сбера и 4 татнефти, выставляет срок гипотезы на 3 месяца и ожидаемую доходность 15%, по прошествию 3 месяцев гипотиза либо подтверждается(доходность выше или равна), либо оправергается.
Финансовый результат сохраняется для пользователя.  
Пример:
На первом прогнозе пользователь мог заработать 10 000р(его финансовый результат 10 000р), а на втором прогнозе потерять 4 000(теперь его финансовый результат 6 000р).
